Calculando BPM usando Javascript e a API da Web Spotify

Esta demonstração usa o navegador Audio API para determinar o andamento de uma música, processando um trecho de 30 segundos de uma música.

Como funciona:

  1. Procure a faixa em Spotify using their Web API
  2. Use o MP3 de visualização dos anos 30 como fonte para a API de áudio
  3. Aplicar um filtro de graves
  4. Encontre picos
  5. Intervalos de contagem entre picos
  6. Agrupe-os
